Projects

  • Online Voting System - GitHub

    This web-based application streamlines the voting process with a secure system built using Java, Spring Boot, JDBC, HTML, CSS, and MySQL. Users register with identity proof, and after admin approval, they can participate in ongoing elections by casting votes.

    The admin is responsible for managing voter information, election candidates, and ensuring a smooth and secure voting process. The system provides a transparent and reliable platform for fair elections.

    Technologies: Java, Spring Boot, MySQL, HTML, CSS

  • Playing Arena Game

    Developed a turn-based arena game where two players engage in strategic combat with attack and defense mechanics. The game involves managing resources like health and energy, making tactical decisions for optimal moves.

    Players take turns using a dice-based combat system to determine the success of attacks and defenses. Special power-ups and critical hits are included to add variety and unpredictability to gameplay.

    Technologies: Java
